.. _orchard:

Orchard
=======

The orchard is a collection of rows with plants. Each plant has clusters and flowers.
Each class represents a way to procedurally generate the next. Flowers have carry a :ref:`stickbug-pollination <pollination>`.` object for defining how they are pollinated.
